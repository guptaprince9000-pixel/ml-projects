# Study Hours vs Marks Prediction

## Overview
This project predicts student marks based on the number of hours studied using Machine Learning.

## Dataset
The dataset is manually created with two columns:
- Hours studied
- Marks scored

## Machine Learning Model
- Linear Regression

## Libraries Used
- pandas
- scikit-learn

## What I Learned
- How to prepare data for ML
- How Linear Regression works
- Training and predicting using sklearn
- Understanding model predictions
