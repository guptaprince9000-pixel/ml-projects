# Hostel Expense Analysis using Pandas

## Overview
This project analyzes hostel daily expenses using Python and Pandas.

## Dataset
The dataset contains hostel expenses with the following columns:
- Date
- Category (Food, Travel, Gym, etc.)
- Amount
- Payment Mode

## Analysis Performed
- Total spending calculation
- Category-wise spending analysis
- Daily spending trends

## Tools Used
- Python
- Pandas

## What I Learned
- Data loading and cleaning
- GroupBy operations
- Basic data analysis using Pandas
